A web application that uses Google Cloud's Vertex AI Imagen and Veo APIs to generate images and videos from text prompts. Simply enter a text description of the image you want to create, and the app will generate high-quality images using Google's Imagen model. You can then edit images and generate videos from them using the Veo model.

Imagen is Google's text-to-image diffusion model that can generate photorealistic images from text prompts. Veo is Google's video generation model that can create videos from images and text prompts. This app provides a user-friendly interface to interact with both APIs through Vertex AI.
